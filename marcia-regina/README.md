# 🏠 Marcia Regina — Real Estate Website

> Site imobiliário profissional, responsivo e moderno para a corretora Marcia Regina Pereira, desenvolvido com HTML5, CSS3 e JavaScript puro.

## 📋 Sobre o Projeto

Landing page de alta conversão para corretora de imóveis, com foco em geração de leads via WhatsApp. O site apresenta a profissional, seus imóveis, depoimentos de clientes e canais de contato direto.

## ✨ Funcionalidades

- ✅ **Design responsivo** — adaptado para mobile, tablet e desktop
- ✅ **Header fixo** com navegação suave entre seções
- ✅ **Menu hamburguer** para dispositivos móveis
- ✅ **Animações de scroll** — elementos aparecem ao entrar na viewport
- ✅ **Contadores animados** — exibe imóveis negociados, anos de experiência e clientes
- ✅ **Carrossel de depoimentos** automático com navegação manual
- ✅ **Botão WhatsApp flutuante** — CTA sempre visível
- ✅ **Links diretos** para WhatsApp e Instagram

## 🛠️ Tecnologias

- **HTML5** semântico
- **CSS3** — Variáveis CSS, Flexbox, Grid, animações
- **JavaScript** puro — Intersection Observer API, carrossel, menu mobile
- **Google Fonts** — Playfair Display + DM Sans

## 🎨 Design

- Tema escuro elegante com detalhes em dourado (`#C9A84C`)
- Tipografia premium: Playfair Display (títulos) + DM Sans (corpo)
- Animações de entrada suaves com `IntersectionObserver`

## 🚀 Como Visualizar

Abra o `index.html` diretamente no navegador — não precisa de servidor.

## 📱 Seções

1. **Hero** — Apresentação com CTA principal
2. **Sobre Mim** — Biografia e estatísticas animadas
3. **Imóveis** — Catálogo (em expansão)
4. **Depoimentos** — Carrossel com avaliações de clientes
5. **Contato** — WhatsApp + Instagram
