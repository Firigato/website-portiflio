# 👤 Personal Portfolio Website

> Site de portfólio pessoal desenvolvido com HTML5, CSS3 e JavaScript, com sistema de repositório e categorias de projetos.

## 📋 Sobre o Projeto

Portfólio pessoal de Fábio Firigato — desenvolvedor web e Python. O site apresenta projetos organizados por categoria, com sistema de busca e filtros. Projeto em desenvolvimento ativo.

## ✨ Funcionalidades Planejadas

- ✅ Header com nome e título profissional
- ✅ Sistema de categorias de projetos
- ✅ Campo de busca por projetos
- 🚧 Cards de projetos com preview
- 🚧 Integração com GitHub API
- 🚧 Seção de habilidades técnicas
- 🚧 Formulário de contato

## 🛠️ Tecnologias

- **HTML5** semântico
- **CSS3** — Flexbox, tipografia customizada
- **JavaScript** — dinâmica e interatividade
- **Google Fonts** — Roboto Slab
- **Adobe Fonts** — Typekit

## 🚧 Status

> **Em desenvolvimento ativo** — estrutura base implementada, funcionalidades sendo adicionadas.

Abra o `index.html` diretamente no navegador.
