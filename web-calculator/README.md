# 🧮 Web Calculator

> Calculadora web interativa desenvolvida com HTML5, CSS3 e JavaScript puro, com suporte às quatro operações matemáticas básicas.

## 📋 Sobre o Projeto

Aplicação web que implementa uma calculadora funcional com interface visual clean, demonstrando habilidades com manipulação do DOM, eventos e lógica de programação em JavaScript.

## ✨ Funcionalidades

- ✅ Operações: **adição, subtração, multiplicação e divisão**
- ✅ Display visual para exibição dos cálculos
- ✅ Botão **Clear (C)** para limpar o display
- ✅ Botão de **apagar último dígito** (`<`)
- ✅ Suporte a **números decimais** (ponto flutuante)
- ✅ Interface responsiva

## 🛠️ Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — estilização e layout com Flexbox
- **JavaScript** — lógica de cálculo e manipulação do DOM

## 🚀 Como Visualizar

Abra o `index.html` diretamente no navegador — sem instalação necessária.
